## AWS Cloud Quest: Cloud Practitioner 🎮

AWS Cloud Quest: Cloud Practitioner is a role-playing game by AWS where you learn and apply cloud skills by solving real world challenges for a virtual city. Instead of just watching videos, you actually build solutions in a real AWS environment, which makes the learning stick a lot more. It covers the same core concepts as the AWS SimuLearn: Cloud Practitioner learning path including compute, networking, databases, security, and high availability.
